Fly API

Developed by FOURNET Paul as part of the JAVA I5RIOC project, flight-api is a modern application built on Quarkus - the Supersonic Subatomic Java Framework.

For more information about Quarkus, visit quarkus.io.

This application can be run in dev mode for live coding, packaged with Maven, and even compiled into a native executable. It demonstrates capabilities such as connection to H2 database via JDBC, Hibernate ORM, and RESTEasy Classic for REST endpoints.

To start exploring and contributing to Fly API, refer to the documentation and related guides available.
